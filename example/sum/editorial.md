# 幂次和题解

设模数为 `P = 998244353`。

对每个 `a_i`，需要计算等比数列

`a_i^1 + a_i^2 + ... + a_i^k`。

若 `a_i mod P = 1`，这一项直接等于 `k mod P`。

否则使用等比数列求和公式：

`a_i * (a_i^k - 1) / (a_i - 1)`。

除法在模意义下用乘以逆元实现。因为 `P` 是质数，且此时 `a_i - 1` 不为 `0`，逆元可以用快速幂计算：

`(a_i - 1)^{-1} = (a_i - 1)^{P-2} mod P`。

每个元素需要两次快速幂，复杂度为 `O(n log k)`，内存复杂度为 `O(1)`。
