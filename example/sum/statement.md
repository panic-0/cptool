# 幂次和

给定一个长度为 `n` 的非负整数数组 `a_1, a_2, ..., a_n` 和一个整数 `k`。

请计算

`sum_{i=1}^{n} (a_i^1 + a_i^2 + ... + a_i^k)`

对 `998244353` 取模后的结果。

## 输入格式

第一行包含两个整数 `n, k`。

第二行包含 `n` 个整数 `a_1, a_2, ..., a_n`。

## 输出格式

输出一个整数，表示答案对 `998244353` 取模后的结果。

## 数据范围

- `1 <= n <= 100000`
- `1 <= k <= 100000`
- `0 <= a_i <= 1000000000`

## 样例输入

```text
3 3
2 3 4
```

## 样例输出

```text
137
```

## 样例解释

`2^1 + 2^2 + 2^3 = 14`，`3^1 + 3^2 + 3^3 = 39`，
`4^1 + 4^2 + 4^3 = 84`，总和为 `137`。
